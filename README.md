# Voici un site internet qui parle de la passion automobile !

## Le site comporte plusieurs pages :
- Une page d'accueil avec un sommaire
- Une page dédiée au sujet
- Une galerie photos
- Et enfin une page à propos de moi ;)

J'ai utilisé Grid et Flexbox, j'ai ajouté un responsive plutôt simple avec des médias queries 
pour définir quand est ce que les élements changent.
###
Il comporte également quelques petites animations telles que un effet "hoover" lorsque l'on passe la souris sur un élément il se déplace.
J'ai aussi ajouté la fonctionnalité de "zoom" dans la galerie photos. Je me suis aidé de l'ia pour cette partie pour le fonctionnement de L'input et les labels.
Je lui ai également demandé pour flouter l'image de fond, pour cela il fallait utiliser ::before devant le body pour éviter de flouter toute la page.

J'ai retouché les images de la galerie photo lorsque c'était nécessaire afin de protéger les propriétaires des véhicules.
###
Le site respecte également la norme W3C
