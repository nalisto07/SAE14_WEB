# Voici un site internet qui parle de la passion automobile !

## Le site comporte plusieurs pages :
- Une page d'accueil avec un sommaire
- Une page dédié au sujet
- Une galerie photo
- Et enfin une page a propos de moi ;)

J'ai utilisé Grid et Flexbox, j'ai ajouté un responsive plutôt simple avec des médias queries 
pour définir quand est ce que les élements changent.
###
Il comporte également quelques petites animations tels que un effet "hoover" lorsque l'on passe la souris sur un element il se déplace.
J'ai aussi ajouté la fonctionnalité de "zoom" dans la galerie photo. Je me suis aidé de l'ia pour cette partie pour le focntionnement de L'input et les labels.
Je lui ai également demandé pour flouter l'image de fond, pour cela il fallait utiliser ::before devant le body pour eviter de flouter toute la page.
